# CodSoft AI Internship

A collection of AI projects built during my **AI Internship at CodSoft**. Each task lives in its own folder with its own code and README, covering classic AI fundamentals — from rule-based systems to search algorithms to deep learning.

## 📁 Repository Structure

```
Codsoft_AI/
├── Task1/    # Chatbot with Rule-Based Responses
├── Task2/    # Tic-Tac-Toe AI (Minimax with Alpha-Beta Pruning)
├── Task3/    # Image Captioning AI (CNN + RNN)
└── README.md
```

## 🚀 Tasks

### Task 1 — [Chatbot with Rule-Based Responses](./Task1)
A rule-based chatbot that identifies user intent and responds using predefined rules, built with **if-else logic** and **regex pattern matching** in Python. Includes a bonus browser-based interactive version.

**Key concepts:** pattern matching, conversation flow, basic NLP fundamentals
**Tech:** Python, Regex · (bonus: HTML/CSS/JS)

---

### Task 2 — [Tic-Tac-Toe AI](./Task2)
An unbeatable Tic-Tac-Toe AI agent that plays against a human player using the **Minimax algorithm with Alpha-Beta Pruning**. Verified unbeatable across 200 simulated games (only wins or draws — never a loss).

**Key concepts:** game theory, adversarial search, recursive backtracking, search tree pruning
**Tech:** Python

---

### Task 3 — [Image Captioning AI](./Task3)
Combines computer vision and NLP: a pre-trained **ResNet50** CNN extracts visual features from an image, and an **LSTM decoder** (with a Transformer decoder alternative also implemented) generates a natural language caption describing it. Trained on the Flickr8k dataset.

**Key concepts:** transfer learning, sequence modeling, encoder-decoder architectures, teacher forcing, greedy/beam search decoding
**Tech:** Python, TensorFlow/Keras, ResNet50/VGG16
